## 📚 What I Learned

While building this project, I improved my understanding of:

- Structuring webpages using semantic HTML.
- Creating responsive layouts with Flexbox.
- Positioning elements using CSS.
- Styling cards with shadows and rounded corners.
- Using JavaScript to toggle classes and create interactive components.
- Organizing project files for better maintainability.

Example of the JavaScript used to toggle the share popup:

```javascript
const shareBtn = document.querySelector(".share-btn");
const popup = document.querySelector(".share-popup");

shareBtn.addEventListener("click", () => {
  popup.classList.toggle("active");
});
```
